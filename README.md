**怎么导入 Zarba 材质包**

1. 打开您的我的世界
2. 单击`选项`
3. 单击`资源包`
4. 将 Zarba 材质包拖入窗口或者单击`打开包文件夹`并将 Zarba 材质包复制到该文件夹
