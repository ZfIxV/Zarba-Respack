**怎麼導入 Zarba 材質包**

1. 打開您的我的世界
2. 按一下 `選項`
3. 按一下 `資源包`
4. 將 Zarba 材質包拖入視窗或者按一下`打開包文件夾`並將 Zarba 材質包複製到該文件夾
