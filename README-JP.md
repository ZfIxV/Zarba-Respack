**How to import Zarba Resourcepack**

1. Minecraft を開く
2. クリック `設定`
3. クリック `リソースパツ𠂊`
4. Zarba マテリアルパッケージをウィンドウにドラッグするか、`パックフオルダーを開く`を開き、Zarba マテリアルパッケージをフォルダにコピーします
