**Zarba 소재 가방을 가져오는 방법**

1. Minecraft 열기
2. 클릭 `설정`
3. 클릭 `리소스 팩`
4. Zarba 재료 패키지를 윈도우로 드래그하거나 `팩 폴더 열기`를 클릭하여 해당 폴더로 복사합니다.
